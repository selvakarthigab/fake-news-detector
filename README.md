📰 Fake News Detector

This is a Machine Learning based web application that detects whether a news article is Real or Fake.

🚀 Features

- Detects fake news using NLP
- Built with TF-IDF Vectorizer
- Machine Learning model prediction
- Simple Streamlit UI

🛠 Tech Stack

- Python
- Scikit-learn
- Streamlit
- Pandas, NumPy

🌐 Live Demo

👉 https://fake-news-detector-ejcbpwxy9hu4dx3echptgc.streamlit.app/

📌 How to Run

pip install -r requirements.txt
streamlit run app.py
